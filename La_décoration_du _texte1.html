La décoration du texte

h1 {
text-align: center;
font-family:"Arial Black", Arial,
"Times New Roman", serif;
text-decoration: blink;
}
.souligne { text-decoration: underline; }
.barre { text-decoration: line-through; }
.ligne_dessus {
text-decoration: overline; }

<h1> ne pas manquer !</h1> <p> propriété CSS<em> text-decoration</em> permet de décorer un peu son texte : <br /> <span class="souligne"> en le soulignant</span> <br> <span class="barre">en le barrant </span>...<br > ...ou encore <span class="ligne_dessus"> mettant une ligne au dessus</span>. </p> .
