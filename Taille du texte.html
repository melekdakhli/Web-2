Taille du texte

En pixels :c'est une façon très précise d'indiquer la taille du texte.
p {
font-size:14px;
h1 {
font-size:22px;
/* Paragraphes de 14 pixels */ }
/* Titres de 22 pixels */ }
En pourcentage : Si vous indiquez "100%", le texte aura une taille "normale".
Si vous mettez "130%", le texte aura une taille correspondant à 130% de la taille normale.
